// CLASSIFIED EQUIPMENT
AGENT TOOLS
EVERY B.O.L.D AGENT CARRIES THESE. NO WEAPONS. JUST BRAINS.
// TOOL 001
📓 FIELD JOURNAL
Every agent keeps a small notebook. Write down everything you see — who is bullying, who is being bullied, where, when, and what happened. This is your evidence. Never act without documenting first.
HOW TO USE: Write in code. Use initials, not full names.
// TOOL 002
🤝 THE SIGNAL
The secret B.O.L.D agent hand signal. Cross your index and middle fingers and tap your chest twice. Only B.O.L.D agents know this. If you see another agent in the wild, use it.
CLASSIFIED: Do not show non-agents.
// TOOL 003
📸 EVIDENCE KIT
For cyberbullying: screenshot the post or message. Save the date and time. Do NOT share the screenshot publicly. Send it directly to Director Oliver Vo. Never delete the original.
HOW TO USE: Screenshot → Save → Report to HQ.
// TOOL 004
🗺️ ZONE MAP
Every agent maps out their school. Mark the hotspots where bullying happens most — bathrooms, hallways, lunch, recess. Know your zone. Watch it every day.
HOW TO USE: Draw your school map in your field journal.
// TOOL 005
🧠 THE SCRIPT
What to say when you see bullying happening right now: "Hey — leave them alone." That's it. Simple. Loud enough for others to hear. Then stand next to the person being bullied. You don't need more words than that.
MEMORIZE IT: "Hey — leave them alone."
// TOOL 006
🔐 CODE SYSTEM
Agents communicate in code when texting about missions. GREEN = safe, no bullying spotted. YELLOW = suspicious activity, watching. RED = active bullying, need backup. Report to Director immediately on RED.
CODES: GREEN · YELLOW · RED
